O que é Hardware?

Hardware é a parte física do computador. São todos os componentes que você pode tocar, como o teclado, o mouse e a tela.

Componentes do Computador

  • Processador: O cérebro do computador.
  • Memória RAM: Onde o computador guarda informações temporárias.
  • Placa-mãe: A placa que conecta todos os componentes.
  • Disco Rígido: Onde os dados são armazenados permanentemente.
  • Fonte de Alimentação: Fornece energia para o computador.

O que é Software?

Software é a parte do computador que não podemos tocar. São os programas e sistemas que fazem o hardware funcionar.

Tipos de Software

  • Software de Sistema: Ajuda o hardware a funcionar (ex: Windows, macOS).
  • Software Aplicativo: Programas que ajudam a realizar tarefas específicas (ex: Microsoft Word, jogos).
  • Software de Programação: Ferramentas para criar outros softwares (ex: Python, Java).
